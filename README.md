## Burger Builder App 

This app is a burger ordering app that allows users to build a burger and order it by signing up or Logging in. 

### Tech/Stack

- React.js
- React-Router
- Redux
- Redux-Thunk
- Firebase

### Features

- Signup/Login
- Routing
- CRUD operations 
- Firebase REST API Authentication
